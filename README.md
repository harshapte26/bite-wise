# Bite-Wise

Welcome to **Bite-Wise**!

## Getting Started

Follow these instructions to set up the project locally.

### Prerequisites

You will need to have [Python 3.11+](https://www.python.org/downloads/) installed.
This project uses [uv](https://docs.astral.sh/uv/) for fast project and dependency management.

If you don't have `uv` installed, you can install it via:

```bash
# On macOS and Linux
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Installation

1. **Clone the repository** (or navigate to your project directory):
   ```bash
   cd bite-wise
   ```

2. **Sync the dependencies and create a virtual environment**:
   ```bash
   uv sync
   ```

### Running the App

You can quickly run the main application script using:

```bash
uv run main.py
```
